# AI Voice Note Summarizer

An Android application that converts voice recordings into structured summaries using AI.

## Features
- 🎤 Voice input (Speech-to-Text)
- 🤖 AI-based summarization
- 📄 Clean UI with modern design

## Tech Stack
- Kotlin
- Jetpack Compose
- Android Studio
- Generative AI APIs
